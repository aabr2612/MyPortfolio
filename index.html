My Work

KiteDB

Python || NoSQL DBMS

KiteDB is an educational purpose NoSQL JSON-based database management system. With features like atomic transactions, role-based access control, AES-CBC encryption, and in-memory B-tree indexing for fast query performance, it ensures secure and efficient data management. It has both console and server-based interfaces to interact with the databases, with different user control levels. A MERN-based User Management app was made to test its integration.

Indian Rummy

Python, Flask || Multiplayer Card Game

Indian Rummy is a multiplayer card game, featuring graph-based decision-making by automated players for turn logic and card grouping. It uses core data structures like Doubly Linked Lists, Queues, Stacks, and Hashmaps to manage gameplay, with sequence detection using DFS and BFS. A Flask-based server enables multiplayer interaction via a CLI frontend. The project utilizes custom-built data structures to emphasize their application in real-world scenarios.

Xzamine

HTML, CSS, JS, Python || Legal Tweet Checker

Xzamine is a browser extension that checks tweets for legality under PECA-2016. It adds an analyze button to each tweet, then uses AI models—Gemma3 (locally via Ollama) and GPT-4o-mini (via OpenAI API) to assess sentiment and update the button text to show if the post is legal or illegal. Then it provides a report view option. It gives a detailed report regarding the post with category probabilities, toxic scores, and rationale.
